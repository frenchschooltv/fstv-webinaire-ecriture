Erreur #2

Texte décousu

L'absence de liens logiques entre les idées

Décousu

« Les examens sont importants. Ils permettent d'évaluer les compétences. Beaucoup d'étudiants stressent. Le stress peut affecter les résultats. »

Cohérent

« Les examens sont importants car ils permettent d'évaluer les compétences. Cependant, beaucoup d'étudiants stressent, ce qui peut affecter les résultats. »

Les connecteurs logiques sont le ciment qui lie vos phrases ensemble.
